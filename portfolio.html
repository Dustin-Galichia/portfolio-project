My Toolkit

HTML5

The enginner of a website. It's the blueprint defines the structure and layout of each page, including headings, paragraphs, images, and links. When I use HTML to build my clients' websites, I create the foundation that holds all the content together.

Read More
CSS

The stylist of a website. It's responsible for making everything look good. With CSS, I add color, typography, spacing, and layout to the elements defined in HTML. It's what gives websites their unique look and feel, making them visually appealing and user-friendly.

Read More
JavaScript

The magic wand of a website. It adds interactivity and makes things happen, like validating forms, creating animations, and updating content dynamically. It's what brings websites to life, making them dynamic and engaging.

Read More
Bootstrap

The versatile toolkit that simplifies website development. It offers ready-made components and styles, allowing me to quickly create responsive layouts and stylish elements, saving time and effort in the process.

Read More
Figma

The digital sketchpad of web design. It's a powerful tool that allows me to create, prototype, and collaborate on designs in real-time, all within a web browser. When I use Figma for my clients' projects, I can design website layouts, create interactive prototypes.
